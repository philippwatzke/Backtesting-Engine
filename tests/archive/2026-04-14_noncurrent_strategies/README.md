# Archived Noncurrent Strategy Tests

Diese Tests gehoeren zu Strategien oder Nebenpfaden, die aktuell nicht Referenz sind.

Typische Beispiele:
- KAMA/MACD
- London Fade
- M6A Fade
- VWAP-/Renko-/Tick-Replay-Nebenpfade
- alte ORB-/Macro-ORB-Pfade

Wenn eine dieser Strategien wieder aktiv werden soll, muss sie bewusst aus dem Archiv geholt und wieder in den aktiven Workflow integriert werden.
